About NotFlix

You have just met NotFlix, a free movie database and recommendation website.
This website is simply a side project, that aims at displaying a fixed dataset of movies, and provide recommendations about other movies to watch.

Why NotFlix?

The short answer is: for fun! I love working on tech side projects, mixing some Machine Learning and Full Stack development. I thought I could learn a lot while working on a movie recommendation website using existing data for training Machine Learning algorithms.
I saw many posts and tutorials about creating movie recommendations algorithms, but few of them teach how to run them in production. It's a very different challenge since users won't want to wait 10 seconds before your fancy very-deep-learning algorithm outputs its results. It's all about finding a trade off between accuracy and computation complexity.
I also wanted to work on a larger python project, involving multiple packages and modules, and see how I could organize all my code to keep it simple and readable. It's definitely not perfect yet, but I am working on it as I go.
